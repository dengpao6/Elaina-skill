# 魔法体系与世界观设定

## 世界观概述

《魔女之旅》的世界是一个由众多**独立国家和城邦**构成的大陆，各地拥有迥异的文化、习俗和奇异现象。旅行魔女们游历各地，是连接这些地方的纽带。

## 魔女资格制度

| 等级 | 说明 |
|------|------|
| 魔女学徒 | 尚未持有资格，跟随魔女师父学习 |
| 魔女（持证） | 通过资格考试，获得专属称号，可独立行动 |
| 称号来源 | 由师父或考官赋予，通常与魔法特质或外貌相关 |

- 伊蕾娜的称号「灰之魔女」来源于她**银灰色的头发**。
- 著名魔女会以称号被广泛知晓（如「星尘魔女」→ 师父）。

## 魔法基础

### 施法方式
- **魔法杖（ほうき / 扫帚）**：魔女飞行和施法的核心道具，伊蕾娜的扫帚陪伴她整个旅程。
- **咒文与意志**：魔法的发动依赖施法者的意志与魔力输出，部分高级魔法需要特定咒文。
- **魔力（マナ）**：魔法的能量来源，每位魔女的魔力量与质不同。

### 魔法分类（原著中涉及）
- **飞行魔法**：操控扫帚飞行，是魔女的标配能力。
- **攻击/防御魔法**：基础战斗魔法，伊蕾娜在战斗时有所展示。
- **治愈魔法**：部分魔女擅长。
- **元素魔法**：操控火、水、风、土等自然元素。
- **特殊魔法**：各国或各魔女可能掌握独特魔法（如时间、记忆等，属于剧情核心魔法）。

### 魔法限制
- 魔法**无法逆转死亡**（原著明确设定）。
- 超出魔力上限会导致施法者精力耗尽甚至昏迷。
- 部分禁忌魔法在某些国家是违禁的。

## 旅行魔女的行规

1. **不干涉为原则**：旅行魔女通常是观察者，不随意介入当地事务。
2. **持证旅行**：各国入境可能需要出示魔女证件。
3. **以魔法换报酬**：在旅途中可以通过提供魔法服务维持生计。
4. **记录见闻**：许多魔女有写旅行日记的习惯（如尼可的著作影响了伊蕾娜）。
